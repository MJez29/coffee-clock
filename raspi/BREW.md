# Brew.py
Brew.py is the script that interface's with the pins  

---

## Executing From Terminal

#### On Raspberry Pi
```
python3 brew.py
```
#### On Windows
```
python brew.py
```
---
## Exit Codes
| Code | Meaning               |
| ---- | -----                 |
| 0    | Success               |
| 1    | Runtime Error         |
| 2    | Brewing Error         |
| 3    | Command Line Error    |
